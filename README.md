An electronic textbook on **Unreal Engine 5**, developed as a college course paper.

The application is a web-based learning platform featuring lectures, practical assignments, quizzes, and a role-based authorization system for students and teachers.

---

## ✨ Features

### Student

- Secure authorization.
- Access to lecture materials.
- Practical assignments.
- Interactive quizzes.
- Can complete only quizzes that have been published by the teacher.

### Teacher

- Secure authorization.
- Administrative panel.
- Create, edit, and delete user accounts.
- Edit lecture materials.
- Edit practical assignments.
- Edit quizzes.
- Enable or disable quiz availability for students.

---

The course includes:

- 📖 5 Lectures

## 🎓 About

This project was created as a **college course paper**.

The assignment was to develop an electronic textbook on any topic using any programming language and technologies. The chosen topic is **Unreal Engine 5**, and the project demonstrates the implementation of a complete educational web application with role-based access control.

# 🇷🇺 Русская версия

Электронный учебник по **Unreal Engine 5**, разработанный в качестве курсовой работы для колледжа.

### 👨‍🎓 Ученик

## 🎓 О проекте

Данный проект разработан в качестве **курсовой работы** для колледжа.

По заданию требовалось создать электронный учебник на любую тему с использованием любых технологий. В качестве темы был выбран **Unreal Engine 5**, а результатом стала полноценная образовательная веб-платформа с разграничением прав доступа для учеников и преподавателей.
